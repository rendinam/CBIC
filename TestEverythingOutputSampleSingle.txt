This script tests most of the CBIC functions. It does NOT test: Enter Server Mode, Install Updated Parameters,
and Install Electronic Gain Corrections, and Compose Bunch Pattern, Diagnostic Dump(takes too long), and
Measure 4nS Bunch Currents(can't test run this script with multiple BPMS if this were included)

Parameters: First - InstrumentAllocation to use
            Second(optional) - any other string you want send to cbic, like -d 1 
Custom:mgs9-third
